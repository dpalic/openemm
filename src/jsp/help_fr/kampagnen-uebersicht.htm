Campagnes - Aperçu

Le menu Campagnes vous permet de structurer vos mailings s’ils sont nombreux. Lors de la création d’une campagne, vous pouvez regrouper tous les mailings traitant du même sujet dans un seul fichier. Par exemple, une campagne peut se composer de tous les mailings envoyés aux clients sur une période de plusieurs semaines dans le cadre d’une campagne de promotion. La campagne peut regrouper des mailings basés sur une action ou sur une date ou simplement tous les mailings envoyés dans le cadre de lettres d’information hebdomadaires. Grâce à l’aperçu, vous pouvez visionner toutes les campagnes définies jusqu’à présent. Les dernières campagnes se trouvent au début de la liste.

  • Campagne: le nom de la campagne peut être modifié en cliquant directement sur la campagne ou sur le bouton Édition ().
  • Champ Description: le champ Description vous offre la possibilité d’ajouter un commentaire descriptif à la campagne. Cette description peut porter sur l’utilisation, la signification ou donner des conseils.
  • Pour supprimer une campagne existante, cliquez sur le bouton Supprimer ().
