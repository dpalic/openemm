Création d’un nouveau type de salutation

Vous pouvez choisir de vous adresser de manière formelle ou informelle à vos clients. OpenEMM vous permet de sauvegarder votre formule de politesse et de l'utiliser lors de votre prochain mailing pour vous adresser à vos clients. Le système établit une distinction entre les destinataires masculins et féminins, ainsi que ceux dont le sexe est inconnu.
Cliquez sur l'onglet Paramètres dans la barre de navigation, puis sur le sous-menu Formule de politesse. Une liste de formules de politesse préétablies s’affiche. Pour créer une nouvelle entrée, cliquez sur l’onglet Nouvelle formule de politesse. OpenEMM ouvre une boîte de dialogue avec plusieurs zones de texte.

  • Dans le champ Description, entrez un nom significatif.
  • Entrez la formule de salutation désirée dans le champ Salutation. La base de données de profils enregistre chaque destinataire selon son sexe (champ de genre). Les destinataires masculins ont le genre 0 tandis que les destinataires féminins ont le genre 1. Les destinataires dont le sexe est inconnu se voient attribuer le chiffre 2. Il y a un champ par genre. Lors de l'envoi du mailing, chaque courriel contient une salutation, un titre (si nécessaire) ainsi que le nom de famille.
  • Une fois que vous avez introduit votre texte, cliquez sur le bouton Enregistrer. La nouvelle entrée s’affiche maintenant dans l’aperçu des Formules de salutation. La première colonne contient le numéro d’identification de cette formule de politesse. OpenEMM attribue automatiquement ce numéro d’identification.

    • Le numéro d’identification est utilisé pour sélectionner une formule de politesse et l’inclure dans la liste de diffusion en utilisant la balise AGNITAS [agnTITLE type = xx]. En lieu et place de xx, insérez le numéro d'identification de la formule de politesse.
