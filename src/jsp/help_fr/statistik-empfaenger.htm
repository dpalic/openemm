Statistiques – Statistiques sur les destinataires

Dans la zone Statistiques sur les destinataires, vous pouvez trouver tous les mouvements de données qui ont eu lieu depuis le premier envoi de votre lettre d’information, comme une demande de renseignements générale. Vous pouvez restreindre les enquêtes sur la base des Groupes cibles définis précédemment ou sur la base des listes de diffusion. La liste déroulante Média affiche les destinataires selon le type de média qu’ils préfèrent pour recevoir leur mailing.
La sélection se fait via la liste déroulante. Après avoir cliqué sur Soumettre, l’ensemble des statistiques s’affiche. Vous obtenez une division en deux catégories « État du destinataire » et « Type de courriel du destinataire » et une analyse précise des changements statistiques de l’ensemble de vos abonnés depuis votre premier mailing.

  • Groupe cible: la liste déroulante Groupe cible vous permet de limiter les destinataires dans des groupes cible. Seuls les destinataires qui se trouvent dans le(s) Groupe(s) cible(s) spécifique(s) seront exportés dans les statistiques. Le paramétrage par défaut est « Toutes les listes de diffusion ».
  • Liste de diffusion: cette liste déroulante vous permet de limiter les destinataires qui recevront le mailing. Seuls les destinataires qui se sont inscrits à la liste de diffusion seront exportés dans les statistiques.
