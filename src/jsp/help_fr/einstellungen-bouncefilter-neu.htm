Paramètres – Nouveau filtre de rebonds

Cliquez sur « Paramètres » situé dans la partie gauche d’OpenEMM. Dans le sous-menu, choisissez « Filtres de rebonds ». Sur le côté droit s’affichent les filtres de rebonds existants. Cliquez sur « Nouveau filtre de rebonds » qui se trouve au début de la liste.

  • Nom: le champ « Nom » s’affiche dans la liste des filtres de rebonds.
  • Description: introduisez une description dans ce champ.
  • Transfert de courriels entrants : activez le champ « Transfert de courriels entrants » et saisissez une adresse électronique dans « Adresse » si vous désirez transférer les courriels entrants à cette adresse.
  • Transfert d’adresse: l'adresse de transfert doit être une adresse électronique régulièrement consultée permettant une réaction rapide à des courriels importants.
  • Réponse automatique: activez le champ Réponse automatique et modifiez les valeurs des champs « Adresse de l’expéditeur », « Objet », « Version texte » et « Version HTML » afin de définir une réponse électronique automatique.
  • Objet: introduisez un texte pour la ligne d'objet du mailing automatique.
  • Version texte: insérez ici le contenu des mailings automatiques au format texte.
  • Version HTML: insérez ici le contenu des mailings automatiques au format HTML.
