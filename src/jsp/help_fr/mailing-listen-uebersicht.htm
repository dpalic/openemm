Listes de diffusion – Aperçu

Lorsque vous vous enregistrez pour la première fois, il n’existe aucune liste de diffusion. La liste de diffusion permet de gérer toutes les déclarations d’acceptation (ou les données de permission) données par les destinataires lors de la demande de courriels. Vous devrez créer une liste de diffusion individuelle pour chaque lettre d’information. Vous pouvez éditer les listes de diffusion existantes en cliquant sur la liste de diffusion désirée ou sur le bouton Édition. Si vous désirez supprimer une liste de diffusion, allez dans la section Aperçu des listes de diffusion. Vous avez alors deux options : cliquez sur le symbole Supprimer situé, dans l’Aperçu, à la fin de la liste de diffusion que vous désirez supprimer ou cliquez sur le nom de cette liste de diffusion. Ceci fera alors apparaître une boîte de dialogue. Cliquez sur le bouton Supprimer de cette boîte de dialogue.
