Modèles - Détails

Cliquez sur le bouton Modèles de la barre de navigation pour afficher un aperçu de tous les modèles disponibles. La liste Aperçu affiche un bouton Édition qui vous permet d’éditer un modèle existant. Une fois que vous avez introduit vos données, vous pouvez sauvegarder le modèle en cliquant sur le bouton Enregistrer. Utilisez le bouton Supprimer pour supprimer le modèle désiré. Les zones de texte et les menus déroulants vous permettent d’effectuer les opérations suivantes :

  • Nom:dans cette zone de texte, vous pouvez entrer un nom significatif pour votre modèle. Ce nom s’affichera plus tard dans les listes d’aperçu et les listes déroulantes utilisées pour sélectionner les modèles.
  • Description: Le texte ne doit comporter qu’une ou deux phrases. Une version raccourcie de la description apparaîtra dans l’aperçu et dans la vision détaillée des modèles.
  • Liste de diffusion : Choose which mailing list will be used with this template.
  • Campaign:: Choisissez la liste de diffusion qui sera associée à ce modèle. Campagne : Choisissez la campagne qui sera associée à ce modèle.
  • Type de mailing :Choisissez entre mailings normaux, mailings basés sur la date ou mailings basés sur une action.
  • Groupes cibles: Affiche le(s) groupe(s) cible(s) spécifique(s). La valeur par défaut est « Tous les destinataires ».
  • Ajouter: si vous désirez associer plusieurs groupes cibles, sélectionnez les groupes cibles désirés dans la liste déroulante et cliquez sur Ajouter. Si plus de deux groupes cibles individuels sont impliqués, tous les destinataires doivent se trouver dans certains groupes cibles (opérateur logique OU). Vous pouvez supprimer un groupe cible en cliquant sur le bouton Supprimer.
  • Objet: Le champ Objet sera la référence qui apparaîtra dans l’en-tête du Courriel du destinataire. Remarque : il est possible de créer des en-têtes d’objet personnalisées via les balises AGNITAS.
  • E-Mail de l'expediteur: il s’agit de l’adresse électronique affichée dans le champ « de » du destinataire. Vous devez utiliser une adresse existante. Le destinataire fait souvent confiance à des adresses comportant un vrai nom comme tom.stanfee@company.com.
  • E-Mail de réponse: il s’agit de l’adresse utilisée pour répondre. Vous devez utiliser une adresse significative et existante comme, par exemple, newsroom@yourcompany.com.
  • Nom de l’expéditeur : il est possible d’attribuer plusieurs noms à l’adresse « real name » de l’expéditeur (Nom complet de l’expéditeur) et à l’adresse pour la réponse (Réponse au Nom complet) qui se différencieront par le nom affiché. Les différents noms peuvent être saisis dans le modèle dans les champs respectifs Nom complet de l’expéditeur et Nom complet pour la réponse. Si vous n’entrez qu’un seul nom, par exemple dans le champ Nom complet de l’expéditeur, celui-ci sera utilisé comme Nom complet pour la réponse.
  • Jeu de caractères:la liste déroulante Jeu de caractères présente le jeu de caractère codant pour le Courriel. Le jeu de caractères par défaut est l’ISO 8859-1 qui est adapté à toutes les langues d’Europe Occidentale.
  • Saut de ligne:La liste déroulante Saut de ligne après est une option qui s’applique à la version texte de votre mailing. Contrairement aux mailings HTML, où les sauts de ligne s’effectuent automatiquement, il faut les définir dans les mailings au format texte. Par défaut, un saut de ligne sera inséré tous les 72 caractères. Vous ne devez pas spécialement modifier ce paramètre.
  • Format: la liste déroulante vous permet de sélectionner le format du courriel. Il existe trois options possibles : format texte ; format texte et HTML ; format texte, HTML et HTML Offline. Au format Texte, tous les Courriels clients pourront ouvrir le mailing mais les options de formatage (gras, hyperlien, taille de la police, …) ne seront pas disponibles. Au format HTML, les images et les options de formatage peuvent être ajoutées au mailing. En utilisant les images des serveurs externes AGNITAS, vos clients pourraient rencontrer des problèmes avec certaines interfaces de navigateurs comme, par exemple, les comptes courriels GMX. Au format HTML Offline, les images seront en fait intégrées comme pièces jointes et les images de la version HTML se téléchargeront une fois le courriel ouvert.
  • Calcul du taux d’ouverture: une option « traceur de pixel » est insérée au Courriel afin de calculer le nombre de mailings HTML ouverts. Lorsque la lettre d’information est ouverte, le serveur charge le pixel et enregistre alors un clic. Vous devez définir si le nombre de clics sur la page doit être enregistré et la manière dont l’enregistrement doit se faire. Le traceur de pixel sera ensuite intégré automatiquement au début ou à la fin du Courriel selon le choix que vous aurez effectué. Si vous l’insérez au début : le serveur enregistre un clic dès que le Courriel est ouvert. Le clic sera enregistré, même si le destinataire ferme ou supprime le message immédiatement après l’avoir ouvert et que les images n’ont pas été totalement téléchargées. Si vous l’insérez à la fin : le serveur enregistre un clic une fois que le courriel a été ouvert et que les images ont été complètement téléchargées. C’est la méthode classique, les chiffres seront légèrement plus faibles que si vous insérez le traceur de pixel au début du Courriel.
    Il est également possible d’insérer le traceur de pixel manuellement via une balise AGNITAS (méthode expliquée dans l’annexe de ce manuel).
  • Version Texte:dans cette zone de texte, vous remplissez la version texte du modèle. Même si vous avez opté pour une version HTML, vous devez remplir ce champ. Ce texte s’affichera si le destinataire ne peut ouvrir la version HTML.
  • Version HTML:Dans cette zone de texte, vous devez remplir la version HTML du modèle.
  • Enregistrer:Vous pouvez facilement enregistrer le modèle en cliquant sur le bouton Enregistrer.
  • Copie:Tous les paramètres du modèle sont copiés et sont absolument identiques. Le mot Copie a été ajouté au nom du modèle et le champ Description est vide. Vous pouvez maintenant donner une description et un nom significatif à votre nouveau modèle et cliquer sur le bouton Enregistrer.
  • Supprimer: En cliquant sur le bouton Supprimer, le modèle sera supprimé du système.
