Mailings – liens traçable

Afin de vous permettre d’évaluer le succès remporté par un mailing, vous pouvez insérer un suivi de liens dans le texte. Dès que le destinataire du mailing clique sur ce lien, cela ouvre un lien URL directement lié au serveur OpenEMM. Le serveur enregistre un clic et l’attribue à un destinataire. Vous pouvez utiliser cette option pour les mailings au format texte et au format HTML. Toutefois, la différence réside dans la notation. L’exemple contient un lien intégré. Un suivi de lien permet d’enregistrer si le destinataire a cliqué ou non sur ce lien. Vous pouvez alors savoir si le destinataire a été intéressé par le sujet du mailing.

  • URL: ce champ montre les adresses Web dans le mailing. Il faut que ce soit une adresse valide vers n’importe quel serveur. Pour vérifier ce lien, cliquez sur la flèche se trouvant devant la zone de texte. Le navigateur ouvre une nouvelle fenêtre/onglet avec l’URL que vous avez définie.
  • Description: Vous pouvez insérer ici une brève description de votre lien. Le destinataire du mailing ne peut voir cette description. Dans l’évaluation statistique, OpenEMM affiche toutefois la description, ce qui simplifie l’évaluation lorsqu’il s’agit de longues URL.
  • Traçable:
    • Non-traçable: le lien apparaît inchangé dans le mailing. Le destinataire peut cliquer sur le lien mais OpenEMM ne peut analyser cette action.
    • Texte seul: avec cette option, le suivi du lien n’est plus actif dans la version html. Il s’avère donc inutile pour les statistiques. Il ne faut utiliser cette option que si un lien spécifique doit apparaître dans le mailing, par exemple, l’adresse électronique de l’entreprise.
    • Texte et HTML: le suivi du lien est actif dans tous les mailings. Il est généralement recommandé de choisir cette option.
  • Action: cette colonne montre les réactions d’OpenEMM lorsqu’un utilisateur clique sur un suivi de lien. Cette action doit être préalablement définie dans la catégorie Actions.
