Destinataire - Exportation

La barre de navigation possède une fonction Exportation à laquelle on peut accéder via l’onglet Destinataire. L’interface se divise en trois zones. Dans la première zone, vous devez choisir les destinataires à exporter. Vous pouvez établir une différence entre la liste de diffusion, le type de destinataire, le statut du destinataire et le groupe cible. Vous pouvez également utiliser tous ces critères en même temps : OpenEMM exporte alors tous les destinataires qui correspondent aux critères sélectionnés

  • Liste de diffusion: toutes les listes de diffusion que vous avez créées s’afficheront. Vous pouvez exporter une liste spécifique en cliquant sur la case à cocher correspondante.
  • Type de destinataire: seuls les destinataires des types choisis dans la liste de diffusion seront exportés. Vous pouvez choisir parmi les types de destinataires suivants :
    • Tous : aucune restriction
    • Destinataire normal: tous les destinataires qui devraient recevoir une lettre d’information ou un mailing spécifique sont repris sous la catégorie de destinataires normaux.
    • Administrateur: les personnes qui doivent vérifier les mailings avant qu’ils ne soient envoyés sont appelées Administrateur. Seuls les administrateurs seront exportés si vous sélectionnez ce type.
    • Destinataire de test: si vous désirez tester vos mailings avant de les envoyer, vous devez inévitablement envoyer des courriels tests pour tester la liste de diffusion. Ces destinataires seront exportés en sélectionnant ce type.
  • Statut du destinataire: seuls les destinataires de la liste de diffusion du statut sélectionné seront exportés. Il existe cinq possibilités :
    • Tous: aucune restriction
    • Actif: le destinataire s’est inscrit à la liste de diffusion spécifique. Vous pouvez voir dans « Remarque » la raison pour laquelle le destinataire a obtenu ce statut.
    • Retourné: l’adresse n’est pas valable, en mode vacances ou le destinataire s’est déconnecté.
    • Désinscrit par admin: un administrateur a désinscrit le destinataire.
    • Désinscrit par destinataire: le destinataire s’est lui-même désinscrit.
  • Groupe cible : seuls les destinataires du groupe cible spécifique seront exportés.

La deuxième zone définit les champs qui seront exportés avec chaque destinataire. En cochant les champs spécifiques, chaque champ créera sa propre colonne dans le fichier CSV final. Vous pouvez également décider si vous voulez voir apparaître le fait que le destinataire s’est inscrit/désinscrit d’une liste de diffusion de la liste.
La troisième zone définit le format d’exportation du fichier. Vous obtiendrez toujours un fichier texte qui se transformera en un fichier Excel normal lors de l’importation du fichier CSV dans Excel. Les menus déroulants suivants signifient :

  • Séparateur: si vous conservez les données des destinataires dans un tableur comme Excel et que vous les sauvegardez dans un fichier CSV, les données sont généralement séparées par un point-virgule. D’autres programmes peuvent recourir à des séparateurs différents. Vous pouvez définir un séparateur différent dans la liste déroulante.
  • Caractère de reconnaissance du texte: si les données du destinataire contiennent le séparateur que vous avez défini, il doit être identifié par un caractère différent. Par exemple, si le séparateur est une virgule et que les données du destinataire contiennent également des virgules, les virgules utilisées comme séparateurs doivent être identifiées séparément avant l’importation.
  • Jeu de caractères:le jeu de caractères par défaut est le jeu de caractère ISO 8859-1 d’Europe Occidentale que vous pouvez également utiliser. Vous pouvez également sélectionner un jeu de caractères différent si votre fichier CSV reconnaît ce jeu de caractères.
