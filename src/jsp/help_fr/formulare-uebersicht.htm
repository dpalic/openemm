Formulaires - Aperçu

Du point de vue du client, la courtoisie est le maître mot du marketing orienté client. Celle-ci débute avec l’inscription à la lettre d’information. OpenEMM vous offre un service personnalisé qui répond immédiatement aux instructions et attentes du client et du prospect quant à l’adhésion ou l’exclusion, et qui leur permet de modifier les données de leur profil.

Les formulaires offrent une interface client flexible dans OpenEMM. Les clients peuvent s’inscrire eux-mêmes, s’abonner à une liste de diffusion, ou modifier les données personnelles de leur profil - comme par exemple le format de courriel désiré- directement sur le site. La création d’un formulaire se compose toujours d'une action de départ, d’un formulaire de réussite, d’un formulaire d’erreurs et d’une action de conclusion.
Pour voir les détails d'un formulaire, cliquez sur le bouton Éditer. Pour supprimer un formulaire, cliquez sur le bouton Supprimer.

  • Nom: définit les noms de formulaires. Ce nom apparaît plus tard dans OpenEMM, où vous pourrez choisir un formulaire.
  • Description: vous donne la possibilité d'ajouter des commentaires descriptifs au formulaire.
