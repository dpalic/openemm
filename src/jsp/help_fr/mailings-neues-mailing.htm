Mailings - Nouveau Mailing

Cette section vous permettra de créer un nouveau mailing. Il vous faut une liste de mailings existante pour pouvoir créer un nouveau mailing. En cliquant sur le bouton Enregistrer, vous créerez un nouveau mailing. Le bouton Supprimer réinitialise toutes les entrées ou supprime le mailing. Le champ doit comporter les informations suivantes :

  • Nom: le nom du Mailing que vous choisissez lors de sa création.
  • Description: Vous pouvez utiliser le champ Description pour ajouter des informations supplémentaires vous permettant de distinguer votre mailing selon les procédures internes.
  • Liste de diffusion: une Liste de diffusion doit déjà exister. Vous commencerez par spécifier la liste de diffusion sur laquelle doit reposer votre mailing. En d’autres termes, vous sélectionnerez les destinataires dont le profil comporte cette liste de diffusion.
  • Campagne: En sélectionnant une Campagne, vous attribuez le mailing à une campagne particulière. Les mises à jour du mailing sont alors plus faciles si vous devez gérer beaucoup de mailings. En outre, cela débloque une autre option d’évaluation statistique. Toutes les campagnes préalablement définies s’affichent dans la liste déroulante. Vous n’êtes pas obligé d’attribuer ce mailing à une campagne. Il peut également fonctionner comme action autonome. Vous pouvez toujours sélectionner une campagne par la suite ou modifier plus tard la campagne à laquelle votre mailing a été attribué.
  • Type: Dans la liste déroulante Type de mailing, sélectionnez Mailing normal. Ceci crée une simple lettre d’information. Vous pouvez également créer un courriel basé sur une action ou sur une base de données.
  • Modèle: vous pouvez attribuer un modèle défini par l’utilisateur au mailing via la liste déroulante.
  • Groupe cible
    • La zone fonctionnelle Groupes cibles contient différentes options permettant de réduire encore plus les destinataires de votre liste de diffusion. Afin de pouvoir travailler avec des groupes cibles, vous devez d’abord définir ces groupes cibles. Si vous désirez envoyer le mailing à plusieurs groupes cibles, sélectionnez les groupes cibles désirés dans la liste déroulante et cliquez sur Ajouter. Si cela concerne plus de deux groups cibles individuels, l’utilisateur peut choisir si tous les destinataires doivent se trouver dans CERTAINS groupes cibles (opérateur logique OR) ou dans tous les groupes cibles (opérateur logique AND). Si vous désirez supprimer de la liste des individus liés à des groupes cibles, cliquez simplement sur le bouton Supprimer.
  • Section E-Mail:
    • Objet : Saisissez le texte qui apparaîtra comme objet du mailing dans le client courriel du destinataire.
    • E-Mail de l'expéditeur is the address shown as sender of the E-Mail. You should therefore use a meaningful address, such as newsroom@yourcompany.com, which does in fact exist.
    • Réponse Email il s’agit de l’adresse électronique utilisée pour les réponses. Il est possible d’attribuer différents noms au Nom complet de l’expéditeur et au Nom complet pour la réponse. Si vous n’entrez qu’un seul nom, par exemple dans le champ Nom complet de l’expéditeur, ce dernier sera utilisé comme Nom complet pour la réponse.
    • Format: dans ce menu, vous pouvez sélectionner le format de courriel que vous désirez envoyer à vos destinataires :
      • Au format Texte, tous les clients courriels pourront ouvrir ce message mais les options de formatage (gras, hyperlien, taille de la police …) ne seront pas disponibles.
      • Au format HTML, il vous est possible d’ajouter des images et un formatage spécifique au message. Si vous utilisez des images des serveurs externes AGNITAS, vos clients peuvent rencontrer des problèmes avec les interfaces de certains navigateurs, notamment les comptes courriels GMX.
      • Au format HTML Offline, les images sont en fait intégrées dans le mail comme pièces jointes tandis qu’en version HTML normale, les images sont téléchargées à partir du serveur une fois que le courriel est ouvert.
  • Jeu de caractères: le jeu de caractères par défaut est le jeu de caractère ISO 8859-1 d’Europe Occidentale que vous pouvez également utiliser.
  • Saut de ligne: cette boîte est utilisée pour définir les sauts de ligne de la version texte de vos mailings. Par défaut, un saut de ligne se place tous les 72 caractères.
  • Le Calcul du taux d’ouverture vous permet de savoir combien de mailings HTML ont été ouverts et lus grâce à ce qu’on appelle un « traceur pixel » inséré dans le courriel. Vous pouvez placer le traceur de pixel au début ou à la fin du courriel.
  • Modèle: Un mailing peut être lié à un modèle. Vous ne devez pas passer par l’onglet Modèles pour voir le contenu du modèle.
