Statistiques – Vue-Domaine

Enfin, vous avez la possibilité d'afficher tous les destinataires selon leur fournisseur de services ou leur domaine d’adresse électronique. Il pourrait s’avérer intéressant de savoir combien de clients possèdent une adresse électronique chez BT, GMX ou AOL, ou si beaucoup de destinataires travaillent pour une même entreprise comme Siemens ou BMW. Vous désirez peut-être créer un mailing pour ce groupe de destinataires.
Les adresses électronique présentent un format commun : name@domain.de. L’aperçu du domaine coupe la première partie de l’adresse jusqu’au caractère @. Il ne vous reste plus que le nom de domaine. OpenEMM compte le nombre de destinataires ayant le même nom de domaine ainsi que le total pour chaque nom de domaine.

  • Groupe cible: la liste déroulante Groupe cible vous permet de limiter les destinataires dans des groupes cibles. Seuls les destinataires qui se trouvent dans le(s) Groupe(s) cibles spécifique(s), seront exportés dans les statistiques.
  • Liste de diffusion: cette liste déroulante vous permet de limiter les destinataires de mailings. Seuls les destinataires qui se sont inscrits à la liste de diffusion seront exportés dans les statistiques.
