Paramètres - Type de formule de politesse

Vous pouvez vous adresser à vos clients soit de manière formelle soit de manière plus informelle. Le système établit une distinction entre les destinataires masculins et féminins, ainsi que ceux dont le sexe est inconnu.

TL’Aperçu établit la liste tous les types de salutations que vous avez déjà définis. En cliquant sur le bouton Éditer, vous pouvez voir tous les détails d’un type de salutation et les éditer. Si vous souhaitez supprimer un type spécifique de salutation, il vous suffit de cliquer sur le bouton Supprimer qui se trouve au bout de la ligne. En confirmant la requête, ce type sera supprimé du système.

  • ID: Le numéro d’identification est un label pour un type de formule de politesse spécifique. Le numéro d’identification est utilisé pour sélectionner une formule de politesse et l’inclure dans la liste de diffusion en utilisant la balise AGNITAS [agnTITLE type = xx]. En lieu et place de xx, inscrivez le numéro d'identification de la formule de politesse.
  • Exemple (ID 1): [agnTITLE type=1] produit le résultat suivant pour le destinataire John Smith : Cher Monsieur Smith
