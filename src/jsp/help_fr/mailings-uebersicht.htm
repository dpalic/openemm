Mailings - Aperçu

Tous les Mailings existants sont triés par date d’envoi (la plus récente en premier). Vous pouvez éditer ce Courriel particulier en cliquant sur son nom ou sur le bouton Édition. Le bouton Supprimer supprime le mailing du système.

  • Mailing: nom du Mailing que vous choisissez lors de sa création.
  • Description: Vous pouvez utiliser le champ description pour ajouter des informations supplémentaires vous permettant de distinguer votre mailing selon les procédures internes.
  • Liste de diffusion: une liste de diffusion doit déjà exister. Vous commencerez par déterminer la liste de diffusion sur laquelle se basera votre mailing. En d’autres termes, sélectionnez les destinataires qui ont cette liste de diffusion dans leur profil.
  • Date d’envoi :date à laquelle le Mailing a été envoyé. Si ce champ est vide, c’est que le mailing n’a pas encore été envoyé.
