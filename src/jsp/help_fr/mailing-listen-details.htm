Listes de diffusion - Détails

Les listes de diffusion constituent l’élément clé de l’envoi de lettres d’information via OpenEMM. Une liste de diffusion comprend tous les prospects et les abonnés d’une lettre d’information ou de mailings individuels. La liste de diffusion gère toutes les déclarations d’acceptation (ou les données de permission) fournies par les destinataires lors de la demande de courriels. Il faut créer une liste de diffusion individuelle pour chaque lettre d’information. Les champs signifient ce qui suit :

  • Nom: chaque fois que vous sélectionnez une liste de diffusion, OpenEMM affichera ce texte que les destinataires ne verront pas.
  • Description: vous devez essayer de donner une description significative qui vous permettra de vous faciliter la tâche lorsque vous devrez gérer de nombreuses listes de diffusion. Avoir des listes de diffusion dont les noms sont clairs constitue un atout, en particulier si plusieurs employés utilisent OpenEMM.
