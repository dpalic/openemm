Statistic - Recipient statistic details

Les fonctions statistiques novatrices d’OpenEMM vous donnent un aperçu détaillé qui vous permet de vérifier rapidement et facilement l'état actuel des destinataires individuels (ou d’un groupe de destinataires). Le fonctionnement de cette fonction est la suivante : Internet est un média rapide, qui change quotidiennement. Il en va de même pour les adresses électroniques. Les changements dans les bases de données d’adresses électroniques sont fréquents, particulièrement juste après un mailing. Il est important de prendre note de ces changements. Les statistiques détaillées enregistrent quotidiennement des inscriptions et des exclusions ainsi que des rebonds, tels que des courriels non-envoyés. Cela vous permet non seulement d'évaluer le succès d'un mailing mais également de vous aider à gérer votre base de données d'adresses. En outre, le délai de contrôle peut être adapté aux besoins personnels. Afin de vérifier les modifications, sélectionnez une date à partir de votre Analyse détaillée et cliquez sur un des numéros.

  • Liste de diffusion: seules les statistiques des destinataires présents dans la liste de diffusion spécifique seront affichées.
  • Groupe cible: en choisissant l’option Groupe cible, vous choisissez le groupe de destinataires pour lequel vous souhaitez obtenir des statistiques. Vous pouvez même comparer les statistiques de 5 groupes cibles maximum via l'ajout de plusieurs Groupes cibles par un simple clic sur le bouton Ajouter.
Vous observez les statistiques :
  • Le statut du destinataire: montre le nombre d’utilisateurs désinscrits, les adresses électroniques non valides ou temporairement indisponibles (Rebonds) et les destinataires actifs.

Types de courriels pour les destinataires :
  • Texte: format ne comprenant pas d’options de formatage (gras, hyperliens, taille de la police...).
  • HTML: dans ce format, les images et le formatage spécifique peuvent être ajoutés au mailing. En utilisant des images des serveurs externes AGNITAS, votre client peut rencontrer des difficultés avec l’interface de certains navigateurs, comme par exemple les comptes GMX.

Analyse des informations: c’est ici que vous pouvez voir toutes les analyses mensuelles, listées par jour. Si vous voulez en savoir plus sur une valeur spécifique, cliquez sur celle-ci. OpenEMM affichera alors plus de statistiques sur cette valeur. OpenEMM vous permet de voir quel destinataire s’est inscrit/désinscrit dans les derniers jours. Avec les flèches gauche et droite, vous pouvez voir les statistiques archivées des derniers mois et des mois à venir.

Vous pouvez exportez ces statistiques en cliquant sur le bouton Exportation. OpenEMM vous permet d’enregistrer les analyses sur votre disque dur au format CSV.
