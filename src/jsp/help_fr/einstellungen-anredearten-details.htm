Paramètres – Détails des Type de salutation

Cliquez sur l'onglet Paramètres dans la barre de navigation, puis sur le sous-menu Formules de salutation. Une liste de formules de politesse prédéfinies s’affiche. Pour éditer une entrée existante, cliquez sur le bouton Éditer. Les zones de texte signifient ce qui suit :

  • ID: OpenEMM attribue automatiquement ce numéro d’identification. Le numéro d’identification est utilisé pour sélectionner une formule de politesse. Utilisez la balise AGNITAS [agnTITLE type = xx] pour inclure cette formule de politesse dans le mailing. Au lieu de xx, entrez le numéro d'identification de la formule de politesse désirée. Les numéros d’identification ne peuvent plus être modifiés par la suite.
  • Description: entrez un nom significatif dans le champ Description.
  • Formule de politesse: (Genre=0, masculin) insère la formule de politesse pour les destinataires masculins. Exemple : Cher Monsieur. Le nom de famille peut être ajouté plus tard dans la définition d'un nouveau mailing.
  • Formule de politesse: (Genre =1, féminin) insère la formule de politesse pour les destinataires féminins. Exemple: Chère Madame. Le nom de famille peut être ajouté plus tard dans la définition d'un nouveau mailing.
  • Formule de politesse: (Genre =2, inconnu) insère la formule de politesse des destinataires de sexe inconnu. Exemple : Cher client.
