Modèles – composantes graphiques

Après avoir cliqué sur Enregistrer, de nouveaux onglets vont apparaître : Contenu, composantes graphiques, pièces jointes, suivi de lien et envoi du mailing. Vous pouvez ajouter des champs supplémentaires à votre modèle si vous le désirez. L’onglet affiche une liste de toutes les images et de tous les graphiques insérés dans le mailing via la balise . Vous pourrez voir dans la partie gauche l’URL de l’image dans le composant image externe. Sur la droite se trouve un aperçu de votre image :

  • Nouveau composant: définit le répertoire d’image sur votre disque dur. Si vous pouvez voir votre image, c’est que tout est correct. Si ce n’est pas le cas, revérifiez votre URL.
  • Liste d’aperçu: Vous pouvez choisir d’utiliser ou non cet élément dans votre mailing via la liste déroulante. Dans la plupart des cas, Utilisation du composant est le paramétrage à choisir. En cliquant sur le bouton Mise à jour, vous communiquez à OpenEMM que, depuis, vous avez modifié votre fichier sur votre serveur. Remarque : en cliquant sur le bouton Supprimer, le système supprime le composant sans demande de confirmation.
