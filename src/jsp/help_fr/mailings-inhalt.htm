Mailings - Contenu

Dans cette fenêtre, vous gérez le contenu de votre mailing. Le contenu d’un mailing peut varier en fonction des groupes cibles. La liste présente les différentes parties du contenu. Cliquez sur une des parties si vous désirez l’éditer. Si vous désirez éditer le contenu d’un groupe cible spécifique, cliquez sur ce groupe cible.
