Action – Nouvelle action

Une action est utilisée pour définir ce qu’il faut faire lorsqu’un destinataire clique sur un lien. Cela peut, par exemple, provoquer un changement dans le profil utilisateur du destinataire en augmentant la valeur dans le champ Nombre de 1. Une action peut également engendrer l’envoi d’un mailing. Les actions peuvent comprendre plusieurs étapes. Dans ce chapitre, nous allons aborder séparément la modification des profils des utilisateurs et l’envoi d’un mailing afin que les procédures soient bien claires. Dans la barre de navigation, cliquez sur le bouton Actions. Cliquez ensuite sur le sous-menu Nouvelle action de la barre de navigation. OpenEMM va alors afficher une boîte de dialogue dans la zone de contenu.

  • Nom: dans la zone Nom, sélectionnez un nom approprié pour l’action. Ce nom s’affichera à chaque fois qu’OpenEMM vous demandera de choisir une action. Afin de vous faciliter la tâche, ce nom doit être significatif.
  • Description: dans cette zone de texte, vous pouvez donner une description de l’objectif attendu de l’action. Ceci prend tout son sens si plusieurs personnes travaillent avec OpenEMM.
  • Utilisation: la liste déroulante a pour objectif de définir la manière dont l’action doit être utilisée. Si vous définissez l’action comme étant « Liens uniquement », vous vous assurez que l’action en question n’apparaîtra dans les listes déroulantes que lorsque qu’elle sera liée à un lien. Ceci réduit le nombre d’entrées dans les listes déroulantes. Si vous sélectionnez l’entrée « Uniquement pour les formulaires », cette action spécifique n’apparaîtra dans les listes déroulantes que pour les pages de formulaire. Si vous sélectionnez « Liens et formulaires », cette action sera toujours disponible

Ensuite, sélectionnez l’action qui doit être exécutée en premier. La liste déroulante Type vous propose les choix suivants : Modification du profil, Envoi du mailing, Inscription, Confirmation de l’inscription, Chargement des données du profil et Désinscription. Ce chapitre est consacré aux profils. Vous devez donc préalablement sélectionner l’option de Modification du profil. Pour terminer, cliquez sur le bouton Ajouter. Vous pouvez maintenant définir avec exactitude les champs du profil de l’utilisateur qui doivent être modifiés par le système ainsi que la manière dont ils doivent être changés. La liste déroulante du Nom des colonnes contient maintenant tous les champs, internes et définis par l’utilisateur, dans le profil de l’utilisateur.

  • Modification du profil: L’option de Modification du profil affiche les données du profil du destinataire. Dans la liste déroulante Nom de colonnes, sélectionnez les changements de profil comme vous le souhaitez.
    • Opérateur: Dans les liste déroulante suivante, sélectionnez le signe « + » comme opérateur. Entrez le chiffre 1 comme valeur dans la dernière boîte. Ceci a pour effet d’augmenter la valeur dans le champ interest_travel de 1 à chaque fois que quelqu’un clique sur ce lien. Cliquez maintenant sur Enregistrer pour fermer ce champ. Vous pouvez insérer des chiffres et du texte.
  • Envoi d’un mailing basé sur une action: cette étape lance l’envoi d’un mailing préalablement défini au destinataire qui a déjà cliqué sur le lien ou envoyé un formulaire (ceci améliore vos paramètres d’ « Utilisation »).
    • MAILING: définissez la liste déroulante TYPE afin d’envoyer un mailing basé sur une action. Le fait de cliquer sur le bouton Enregistrer envoie vos entrées au système. La liste déroulante affiche toutes les listes de mailing basé sur une action que vous avez définies jusqu’à présent dans Mailings - Nouveau mailing. Sélectionnez le mailing approprié. La liste déroulante ne contient pas les mailings normaux, les mailings qui ne sont pas marqués comme « À envoyer » ou les mailings basés sur la date.
    • DELAY: la liste déroulante Délai établit le laps de temps entre le début de l’action et l’envoi du courriel. Il y a différentes options : d’une heure à sept jours. Il est intéressant de différer un envoi lorsque, par exemple, vous effectuez une offre personnalisée au destinataire. Toute personne normale différerait l’envoi d’au moins quelques heures.
  • Inscription : cette étape édite les informations de la liste des destinataires à partir de la liste de diffusion.
    • Double Inscription: Les paramètres de confirmation d’inscription permettent à l’utilisateur d’éviter les enregistrements erronés, malveillants ou contenant des erreurs. OpenEMM enverra un courriel de confirmation à l’abonné après son enregistrement. Ce message contient un lien sur lequel doit cliquer l’abonné pour confirmer son inscription. Le système n’enregistrera les données dans la base de données que si l’abonné clique sur ce lien. Ce n’est qu’après avoir terminé correctement cette étape de confirmation que l’abonné recevra un message de la liste de diffusion à laquelle il ou elle s’est inscrit(e). Si ce dernier ne réagit par au courriel de confirmation, le profil sera bloqué et automatiquement supprimé d’OpenEMM après trois mois.
      L’option de confirmation d’inscription est sélectionnée par défaut. La « Colonne clé pour la vérification des doublons et mises à jour » définit, par défaut, le courriel comme moyen de confirmation. L’option « Utilisation de la confirmation d’inscription » doit être sélectionnée si l’utilisateur veut être intégré dans la liste de diffusion après confirmation d’inscription.
  • Chargement des données du profil: après avoir cliqué sur le suivi de lien ou après avoir envoyé un formulaire, les données du profil du destinataire seront chargées dans un formulaire. Ceci permet par exemple à l’utilisateur de modifier lui-même le type de courriel en passant du format texte au format HTML.
  • Désinscription: le destinataire sera supprimé de la liste de diffusion s’il clique sur un lien spécifique ou s’il renvoie un formulaire.

En cliquant sur « Enregistrer », vous enregistrez l’action que vous venez de terminer. En cliquant sur « Supprimer » et en acceptant la requête, l’action sera effacée de votre système.
