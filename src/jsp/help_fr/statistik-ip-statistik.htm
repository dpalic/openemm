Statistiques – Statistiques IP

L’objectif des statistiques IP est de déterminer les adresses IP des destinataires utilisées pour s’abonner à une liste de diffusion. OpenEMM montre le top 20 des adresses IP. Cette fonction statistique est destinée à empêcher la fraude. Il existe des organismes de loterie qui enregistrent tous les membres qu’ils peuvent trouver pour les faire participer (en payant) à leurs loteries, tombolas et promotions. Étant donné que les membres ne sont pas vraiment intéressés par la promotion en question, de tels courriels sont inutiles.
Certains organismes utilisent même de fausses adresses ; des courriels envoyés à ces adresses ne seront jamais lus. Il est relativement facile de créer un domaine d’adresse, de sorte que l'adresse électronique elle-même ne révèle pas l’escroquerie. Toutefois, l'organisme a généralement une adresse IP. Si cette adresse IP apparaît brusquement dans le top 20 des statistiques, cela peut indiquer une fraude. Vous pouvez également exporter ces statistiques en cliquant sur le bouton Exportation. Vous obtiendrez un fichier CSV que vous pourrez enregistrer sur votre ordinateur.

L'adresse IP montre l’adresse IP du destinataire. Dans la colonne Destinataires, le système établit la liste des abonnés qui ont utilisé cette adresse pour s’inscrire. Différentes listes déroulantes peuvent être utilisées pour réduire la base de données statistique :

  • Groupe cible: la liste déroulante Cible sélectionne les membres de votre groupe cible.
  • Liste de diffusion: la liste déroulante ne sélectionne que les abonnés d'une liste de diffusion particulière pour l'évaluation statistique. Les deux critères de sélection peuvent être combinés.

    • Pour terminer, OpenEMM va recalculer toutes les statistiques une fois que vous aurez cliqué sur le bouton OK.
