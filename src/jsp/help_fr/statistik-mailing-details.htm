Statistiques – Détails des statistiques sur les mailings

L’aperçu des statistiques vous donne une liste de toutes les lettres d’information envoyées jusqu’à présent, dans la section Statistiques des adresses. Il vous suffit de cliquer sur l'une de ces lettres d'information dans la colonne Mailing pour la sélectionner et l’évaluer. Vous verrez immédiatement, sous forme graphique et numérique, le nombre de clics réalisés sur les liens présents dans votre mailing. La même évaluation est accessible en sélectionnant un envoi de la section Mailings et en cliquant ensuite sur l'onglet Statistiques.

  • Groupe cible: en choisissant Groupe cible, vous sélectionnez les destinataires pour lesquels vous souhaitez obtenir des statistiques. Vous pouvez même comparer les statistiques de 5 groupes cibles maximum en les ajoutant via le bouton Ajouter.
    Des informations supplémentaires sont disponibles sur tous les courriels qui ont été ouverts. Un clic sur le lien Courriels ouverts et OpenEMM affichera les données en fonction de domaines issus des adresses électroniques des destinataires et classera les 20 meilleurs domaines en fonction du nombre de destinataires.
  • URL:le premier passage établit tous les suivis de liens présents dans le mailing. Dans la première colonne, OpenEMM vous montre l'adresse du site Web. Dans la deuxième, vous pouvez voir le nombre de clics enregistrés. En utilisant la flèche située en face de l'adresse du site Web, vous pouvez visiter la page cible dans une nouvelle fenêtre.
  • Total des clics de l’abonné: ce graphique montre la proportion entre les destinataires et les clics.
  • Courriels ouverts :ce résultat est uniquement valable pour les courriels HTML possédant la balise AGNITAS [agnONEPIXEL] incluse dans le module texte HTML.
  • Exclusion: indique le nombre de destinataires qui ont utilisé le lien d’exclusion dans le mailing.
  • Rebonds: cette valeur correspond au nombre de courriels contenant une fausse adresse de destinataire ou d'autres problèmes de réception liés à l'envoi.
  • Destinataire: le nombre de destinataires auxquels les courriels ont été envoyés.

Vous pouvez nettoyez les statistiques pour filtrer ainsi tous les courriels d’administrateur et de test. Le bouton Exportation vous permet d'exporter les statistiques dans un fichier CSV.
