Paramètres – Champs du profil

Les données à caractère personnel de chaque destinataire du mailing sont enregistrées dans un profil à part dans la base de données d’OpenEMM. Pour afficher les détails d'un champ, cliquez sur le nom de ce champ ou sur le bouton Éditer qui se trouve à la fin de la ligne. Le bouton Supprimer supprimera ce champ. OpenEMM vérifie si un élément du programme utilise encore le champ avant de le supprimer. Vous pouvez ne pas supprimer un champ si, par exemple, celui-ci est encore utilisé par un groupe cible. Dans ce cas, vous devez d'abord supprimer le groupe cible.
Les différents champs signifient ce qui suit:

  • Nom du champ: nom du champ, qui apparaît dans le menu Options. Par exemple, dans les définitions des groupes cibles.
  • Nom du champ dans la Base de données (BD): ce nom apparaît seulement dans la gestion des champs. Ce champ donne le nom qui sera enregistré dans la base de données des profils.
  • Type: un champ peut inclure différents types de données. Les différents types de champs suivants sont disponibles :
    • Date: ces champs enregistrent une date. L’entrée doit être du type YYYYMMDD (AAAAMMJJ). Par exemple 20080301 pour le 1er mars 2008.
    • Numérique: ce champ permet de saisir des chiffres.
    • Alphanumérique: aucune restriction ne s’applique lorsque vous choisissez ce type de champ.
