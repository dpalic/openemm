Statistic - Comparaison

Si, toutefois, vous voulez suivre le succès de vos campagnes de mailings en les comparant sur une période plus longue, la représentation statistique des mailings n’est pas très appropriée. OpenEMM de AGNITAS AG vous permet de comparer le succès d'un certain nombre de campagnes de mailings dans un aperçu. En associant plusieurs mailings, vous pouvez suivre le développement de vos activités de mailing en un clin d'œil. Les résultats peuvent également être exportés pour être affichés et traités dans des tableurs. Voici la procédure 

  • Ouvrez l’aperçu de tous les mailings envoyés jusqu’à ce jour en cliquant sur le bouton Statistiques de la barre de navigation.
  • Le sous-menu Comparaison affiche une liste de tous les mailings envoyés jusqu’à ce jour.
  • Sur la droite, sous l’option Comparaison, se trouve une case à cocher pour chaque mailing. Cochez-la pour tous les mailings que vous voulez comparer. Cliquez ensuite sur le bouton Comparaison.
  • Après un instant, OpenEMM affiche une liste avec les colonnes Destinataires, Clics, Ouverts, Rebonds et Exclusion. Cliquer sur le bouton « Retour » vous permet de procéder à une nouvelle sélection.

    • Option supplémentaire de surveillance statistique : la sélection par groupes cibles. Tout ce que vous avez à faire est de sélectionner une entrée dans la liste déroulante Cible et cliquer sur OK pour confirmer. (Vous pouvez également modifier les définitions des groupes cibles dans la vue détaillée pour éviter d’appliquer votre choix à tous les mailings).
