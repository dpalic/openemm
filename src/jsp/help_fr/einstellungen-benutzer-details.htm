Paramètres – Détails d’utilisateur

Pour chaque compte, vous pouvez créer des utilisateurs individuels, en leur affectant des droits d’utilisateur spécifiques. Chaque compte doit disposer d'au moins un utilisateur qui se connecte au moment de l'enregistrement à OpenEMM. Ces utilisateurs sont répertoriés dans Paramètres - Utilisateurs. Là, vous pouvez éditer les détails de l'utilisateur. Une fois que vous avez réalisé toutes vos entrées, cliquez sur le bouton Enregistrer.

  • Nom complet: reprend le prénom et le nom de famille de l’utilisateur ou le nom de l’entreprise.
  • Nom d’utilisateur: nom spécifique saisi par l’utilisateur lorsqu’il se connecte à OpenEMM. Il ne peut contenir d’espaces ou de caractères spéciaux mais peut contenir des chiffres.
  • Mot de passe: saisissez un mot de passe dans la case correspondante. Pour des raisons de sécurité, les navigateurs Web n’affichent pas les caractères du mot de passe mais des étoiles. C’est pourquoi le mot de passe doit être confirmé en le réintroduisant une seconde fois.
  • Langue: vous avez le choix entre la version allemande, anglaise ou française du logiciel.
  • Fuseau horaire: si un utilisateur se trouve dans un fuseau horaire différent, c’est ici qu’il faut le préciser.
  • Client: attribue un compte client à l'utilisateur.
