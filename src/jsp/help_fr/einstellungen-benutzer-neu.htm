Paramètres - Nouvel Utilisateur

Pour chaque compte, vous pouvez créer des utilisateurs individuels, en leur affectant des droits spécifiques d’utilisateur. Chaque compte doit disposer d'au moins un utilisateur qui se connecte au moment de l'enregistrement avec OpenEMM. Via la barre de navigation, ouvrez la boîte de dialogue de saisie de texte pour créer un nouvel utilisateur : allez dans Paramètres, sous-menu Utilisateurs, puis ouvrez l'onglet Nouvel utilisateur dans la zone de contenu.
Complétez les données requises dans les champs. Le Nom complet reprend le prénom et le nom de famille de l’utilisateur ou un nom d’entreprise. Le nom d’utilisateur est un nom spécifique que doit saisir l’utilisateur lorsqu’il se connecte à OpenEMM. Celui-ci ne peut contenir ni espaces ni caractères spéciaux, mais peut contenir des chiffres.
Saisissez un mot de passe dans la case correspondante. Pour des raisons de sécurité, les navigateurs Web n’affichent pas les caractères, mais des étoiles lors de la saisie du mot de passé. C’est pourquoi le mot de passe doit être confirmé en le réintroduisant une seconde fois
Utilisez les différentes listes déroulantes pour entrer les informations suivantes

  • Langue: vous avez le choix entre la version allemande, anglaise ou française du logiciel.
  • Fuseau horaire: si un utilisateur se trouve dans un fuseau horaire différent, c’est ici que vous pouvez le préciser.

    • Une fois vos entrées terminées, cliquez sur le bouton Créer. Lorsque vous avez créé le nouvel utilisateur, OpenEMM vous propose un aperçu de tous les utilisateurs existants dans le système.
