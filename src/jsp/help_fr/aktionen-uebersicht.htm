Actions - Aperçu

Une action est utilisée pour définir ce qu’il convient de faire lorsque le destinataire clique sur un lien. Une action peut également engendrer l’envoi d’un mailing. Les actions peuvent comprendre différentes étapes. L’aperçu permet d’afficher les actions suivies des symboles Supprimer et Éditer. Si vous cliquez sur le premier bouton, l’action affichée sera supprimée. Un clic sur le deuxième bouton permettra d’afficher les détails de l’action que vous pouvez éditer.
