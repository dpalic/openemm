Paramètres – Filtre de rebonds

Le filtre de rebonds définit les actions exécutées après la réception d’un courriel de réponse émanant du destinataire. Lors de la réception d'une réponse, toutes les réponses automatiques sont d’abord supprimées pour raisons de commodité. Les actions des filtres à rebonds s’appliquent aux courriels restants. Vous pouvez alors les transférer ou générer une réponse automatique.
Création d’un nouveau filtre de rebonds
CCliquez sur « Paramètres » dans la partie gauche d’OpenEMM. Dans le sous-menu, choisissez « Filtres de rebonds ». Sur le côté droit s’affichent les filtres de rebonds possibles. Cliquez sur « Nouveau filtre de rebonds » qui se trouve au début de la liste.
Choisissez le filtre de rebonds que vous souhaitez éditer en cliquant sur son nom. La boîte de dialogue de saisie de texte contenant les propriétés du filtre de rebonds s’ouvre.

  • Nom: le champ « Nom » est indiqué dans la liste des filtres de rebonds.
  • Description: Introduisez une description dans ce champ.

Pour terminer, cliquez sur Enregistrer. Ensuite, vous serez redirigé vers la page Aperçu des filtres de rebonds. Une adresse électronique du type xxx@filter.yourcompany.com s’affiche dans l'aperçu. Tous les courriels envoyés à cette adresse seront traités conformément aux actions définies par le filtre de rebonds. Une dernière étape est ensuite nécessaire pour terminer l'installation du filtre de rebonds.
Tous les courriels de réponse à un mailing seront envoyés à l’adresse de réponse de l’expéditeur. L’adresse de réponse du mailing (si elle est définie) sera, par exemple, newsletter@company.com. Définissez maintenant dans le client courriel de cette adresse un transitaire pour l’adresse électronique reprise dans l’aperçu (xxx@filter.yourcompany.com). Ce n’est que si le transitaire a été défini que la gestion du rebond peut filtrer toutes les réponses générées automatiquement et transférer les courriels restants selon les paramètres du filtre de rebonds et/ou y répondre au moyen d’une réponse automatique.
