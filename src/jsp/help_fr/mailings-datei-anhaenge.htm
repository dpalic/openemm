Mailings – Attachement de fichiers

La fonction optionnelle d’attachement de fichiers permet d’améliorer considérablement l’utilité d’OpenEMM. Cette fonction vous permet de fournir de plus amples informations aux destinataires à propos du texte de la lettre d’information, au-dessus de celui-ci. Vous pouvez, par exemple, envoyer des vouchers ou des listes de prix via un fichier attaché.
OpenEMM s’assure également que les fichiers attachés ne contiennent pas de virus ou de vers informatiques et qu’ils ne sont ni corrompus ni malveillants. Chaque fois qu’un fichier est téléchargé sur le serveur OpenEMM, il est préalablement scanné par un logiciel antivirus. OpenEMM n’accepte les fichiers que s’ils ont été scannés et qu’ils ne contiennent pas de virus.

  • Pour joindre un fichier à votre mailing, vous devez d’abord commencer par sélectionner le mailing. Cliquez sur le bouton Mailings de la barre de navigation et cliquez ensuite sur le mailing désiré dans l’aperçu. Dans les détails du mailing, ouvrez l’onglet Pièce jointe.
  • Dans la boîte de dialogue, introduisez le nom et le répertoire du fichier attaché dans la zone de texte Pièce jointe. Il est généralement plus facile de cliquer sur le bouton Parcourir pour ouvrir la boîte de dialogue Windows de sélection de fichiers. Sélectionnez le fichier désiré et cliquez sur Ouvrir. Le navigateur insère alors le répertoire complet du fichier dans le champ.
  • La zone de texte Nom de la pièce jointe est utilisée pour donner un nom à votre fichier attaché. Dans notre exemple, le nom original news.pdf sera utilisé.
  • La liste déroulante Cible vous permet de sélectionner les destinataires qui recevront la pièce jointe. Étant donné que vous pouvez définir un nombre indéterminé de pièces jointes pour chaque mailing, vous pouvez facilement attribuer différentes pièces jointes à différents groupes cibles.

En cliquant sur Ajouter, le fichier sélectionné est joint à votre mailing. Sous la boîte de dialogue de saisie de texte, OpenEMM affiche maintenant une liste de toutes les pièces jointes que vous avez définies. La liste des pièces jointes montre différents détails et possibilités d’interfaces pour chacune des entrées. Si vous désirez supprimer une pièce jointe, cliquez sur le bouton Supprimer. Attention : la pièce jointe sera automatiquement supprimée sans demande de confirmation.

  • Le type MIME (Multipurpose Internet Mail Extensions) vous donne le type de pièce jointe que vous avez sélectionné. Lorsque vous envoyez un mailing contenant une pièce jointe au format texte, cela sera caractérisé par texte / format brut. Dans notre exemple, la pièce jointe est un fichier pdf. Le type MIME sera donc application/pdf. Tous les éléments HTML inclus seront également affichés ici.
    La Taille originale du fichier est différente de la Taille du fichier dans le courriel. En fait, le logiciel convertit la pièce jointe en un format que l’on peut envoyer (format ASCII) : il est 30 ou 40 pour cent plus grand que le fichier original. Les destinataires de courriels clients modifieront la pièce jointe qui reprendra son format original.
  • Une fois de plus, vous avez la possibilité de définir un Groupe cible pour cette pièce jointe. Si vous envoyez plusieurs pièces jointes, il vous sera ensuite plus facile d’attribuer une pièce jointe à un groupe différent. Dans notre exemple, laissez les paramètres d’origine. Toutefois, n’oubliez pas de spécifier au système d’Enregistrer ces modifications.
  • Une fois que vous êtes satisfait des données que vous avez entrées, cliquez sur Enregistrer afin de demander au système d’enregistrer vos données. Si vous désirez supprimer une pièce jointe du mailing, cliquez simplement sur le bouton Supprimer.

Pour définir d’autres fichiers attachés, ouvrez à nouveau la boîte de dialogue et cliquez sur le bouton Ajouter.
