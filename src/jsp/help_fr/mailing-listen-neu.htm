Listes de diffusion – Nouvelles listes de diffusion

Cliquez sur le bouton Mailings situé sur la gauche de la page Web. Dans le sous-menu, sélectionnez l’option Nouvelle liste de diffusion. Une boîte de dialogue s’ouvre sur la droite : vous pouvez l’utiliser pour entrer vos données. Le contenu explicite se trouve déjà dans les champs respectifs afin de vous aider. Entrez les données pertinentes dans les champs Nom et Description. Chaque fois que vous devrez sélectionner une liste de diffusion, OpenEMM affichera ce texte que les destinataires ne verront pas. Vous devriez essayer de recourir à des descriptions claires qui vous faciliteront la tâche lorsque vous devrez gérer plusieurs listes de diffusion. Des listes de diffusion dont les noms sont clairs sont un atout, en particulier si plusieurs employés utilisent OpenEMM.

  • Nom: Chaque fois que vous sélectionnez une liste de diffusion, OpenEMM affichera cette zone de texte. Toutefois, les destinataires ne la verront pas.
  • Description: vous devez essayer de donner une description significative qui vous permettra de travailler plus facilement lorsque vous devrez gérer de nombreuses listes de diffusion. Avoir des listes de diffusion dont les noms sont clairs constitue un atout, en particulier si plusieurs employés utilisent OpenEMM.

Pour terminer, cliquez sur le bouton Enregistrer. Le système enregistre vos données. Le navigateur passe automatiquement à l’écran d’édition des listes de diffusion. Si vous cliquez sur Aperçu dans le menu Listes de diffusion, votre nouvelle liste de diffusion s’affichera.
