Mailings – composantes graphiques

Les courriels en HTML contiennent souvent des illustrations ou des éléments graphiques. Grâce à la balise , vous pouvez insérer un lien vers une image. Elle se trouve généralement sur votre serveur et a une adresse de type: http://www.my-company.com/newsletter/grafiken/unterschrift.gif.Via l’onglet composantes graphiques, vous pouvez vérifier si l’URL a été insérée correctement. Internet Explorer ne peut afficher que les images au format GIF, JPG et PNG. Ouvrez le menu Mailings via la barre de navigation et cliquez sur le mailing désiré. Cliquez sur l’onglet composantes graphiques dans la boîte de dialogue de saisie de texte située sur la droite.

  • Nouveaux composants: l’onglet affiche une liste de toutes les images et de tous les graphiques insérés dans le mailing via la balise . La partie gauche vous montre l’URL de l’image sous Composant image externe. La partie droite vous montre un aperçu de votre image. Si vous pouvez voir votre image, c’est que tout est en ordre. Sinon, revérifiez l’URL.
  • Menu déroulant: grâce à la liste déroulante, vous pouvez sélectionner si vous désirez intégrer ou non cet élément dans votre mailing. Dans la plupart des cas, la fonction Utilisation du composant constitue la bonne option. En cliquant sur le bouton Mise à jour, vous communiquez à OpenEMM que vous avez modifié le fichier sur votre serveur. Si vous désirez supprimer l’image, le système ne vous demandera pas de confirmation une fois que vous aurez cliqué sur le bouton Supprimer.
