Mailings - Mailing

Un mailing permet de relier toutes les informations (destinataires, listes de diffusion, groupes cibles) et d’envoyer des courriels. Les étapes suivantes vous guideront à travers le processus afin de vous montrer comment fonctionne l’envoi d’un mailing :

  • Section Générale
    • Nom: le nom du Mailing que vous avez choisi lors de sa création.
    • Description: vous pouvez utiliser le champ Description pour ajouter des informations supplémentaires vous permettant ainsi de distinguer votre mailing selon les procédures internes.
    • Liste de diffusion: une liste de diffusion doit déjà exister. Vous commencerez par déterminer la liste de diffusion sur laquelle se basera votre mailing. En d’autres termes, sélectionnez les destinataires inscrits à cette liste de diffusion dans leur profil.
    • Campagne: Si vous sélectionnez une campagne, vous attribuez le mailing à cette campagne particulière. Cela facilitera les mises à jour des mailings si vous devez en gérer beaucoup. En outre, cela vous permet d’avoir accès à une option d’évaluation statistique. Toutes les campagnes préalablement définies se trouvent dans la liste déroulante. Vous n’êtes pas obligé d’attribuer le mailing à une campagne. Il peut également fonctionner comme action autonome. Vous pouvez toujours sélectionner une campagne ou modifier l’affectation de campagne de votre mailing plus tard.
    • Type: dans la liste déroulante Type de mailing, sélectionnez Mailing normal. Cette action créera une simple lettre d’information. Vous pouvez également créer un courriel basé sur une action ou sur une base de données.
    • Modèle: il est possible d’attribuer au mailing un modèle défini par l’utilisateur via une liste déroulante.
  • Groupe cible
    • Groupes cibles La zone fonctionnelle contient différentes options permettant de réduire encore plus les destinataires de votre liste de diffusion. Afin de pouvoir travailler avec des groupes cibles, vous devez d’abord définir ces groupes cibles. Si vous désirez envoyer le mailing à plusieurs groupes cibles, sélectionnez les groupes cibles désirés dans la liste déroulante et cliquez sur Ajouter. Si cela concerne plus de deux groupes cibles individuels, l’utilisateur peut choisir si tous les destinataires doivent se trouver dans CERTAINS groupes cibles (opérateur logique OU) ou dans tous les groupes cibles (opérateur logique ET). Si vous désirez supprimer de la liste des individus liés à des groupes cibles, cliquez simplement sur le bouton Supprimer.
  • Section E-Mail
    • Objet : Saisissez le texte qui apparaîtra comme objet du mailing dans le courriel client du destinataire.
    • E-Mail de l'expéditeur correspond à l’adresse qui s’affiche comme étant celle de l’expéditeur du courriel. Vous devez utiliser une adresse significative, comme par exemple newsroom@yourcompany.com, qui est en fait une adresse existante.
    • Nom de l'expéditeur: vous pouvez remplacer l’Adresse électronique par un nom. Ce nom s’affichera comme destinataire du courriel dans le Courriel client du destinataire. L’adresse de l’expéditeur sera toujours disponible, tel que ceci est défini dans l’aperçu du mailing. Il est possible d’attribuer différents noms au Nom complet de l’expéditeur et au Nom complet pour la réponse. Si vous n’entrez qu’un seul nom, par exemple dans le champ Nom complet de l’expéditeur, ce dernier sera utilisé comme Nom complet pour la réponse.
    • Format:
      • Au format Texte, tous les clients courriels pourront ouvrir ce message mais les options de formatage (gras, hyperlien, taille de la police …) ne seront pas accessibles.
      • Au format HTML, il vous est possible d’ajouter des images et un formatage spécifique au message. Si vous utilisez des images des serveurs externes AGNITAS, vos clients pourraient rencontrer des problèmes avec les interfaces de certains navigateurs, notamment les comptes courriels GMX.
      • Au format HTML Offline, les images sont en fait intégrées dans le courriel comme pièces jointes tandis qu’en version HTML normale, les images sont téléchargées à partir du serveur une fois que le courriel est ouvert.
    • Jeu de caractères: le jeu de caractères par défaut est le jeu de caractère ISO 8859-1 d’Europe Occidentale que vous pouvez également utiliser.
    • Saut de ligne: cette boîte de dialogue est utilisée pour définir les sauts de ligne de la version texte de vos mailings. Par défaut, un saut de ligne se place tous les 72 caractères.
    • Modèle: Un mailing peut être lié à un modèle. Vous ne devez pas passer par l’onglet Modèles pour voir le contenu du modèle.
    • Le Calcul du taux d’ouverture vous permet de savoir combien de mailings HTML ont été ouverts et lus grâce à ce qu’on appelle un « traceur pixel » inséré dans le courriel. Vous pouvez placer le traceur de pixel au début ou à la fin du courriel.

fin d’enregistrer vos modifications, cliquez sur le bouton Enregistrer. Pour copier ce mailing spécifique, afin de faciliter la création de nouveaux mailings, cliquez sur le bouton Copier. Le bouton Supprimer efface le mailing du système.
