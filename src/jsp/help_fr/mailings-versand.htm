Mailings - Diffusion

Votre mailing est maintenant prêt : le contenu a été incorporé et les courriels de test ont été révisés plusieurs fois. Vous êtes maintenant prêt à envoyer votre mailing !

  • Cliquez sur le sous-menu Envoi du mailing situé sur l’écran Envoi du mailing. OpenEMM vous permet de voir le mailing en question. La zone de sélection Destinataires affiche tous les destinataires que vous avez définis comme groupe cible avant d’envoyer le mailing afin que vous puissiez revérifier votre sélection. Le système affiche également le nombre de destinataires et les formats de courriel choisis.
  • Dans la zone Heure de diffusion, plusieurs listes déroulantes vous permettent de sélectionner précisément l’heure à laquelle commencera l’envoi du mailing. Vous pouvez choisir un envoi automatique à minuit et OpenEMM répondra à vos souhaits.
  • Cliquez maintenant sur le bouton Envoi afin de commencer l’envoi du mailing. Si vous désirez effectuer de nouvelles modifications, cliquez sur Annuler.

Une fois l’envoi effectué, l’aperçu Envoi du mailing vous affiche un rapport d’état. Toutefois, vous devez cliquer sur l’onglet Envoi du mailing pour générer la création d’un rapport d’état qui sera ensuite affiché.
Vous pouvez encore afficher la prévisualisation et envoyer le mailing à une liste de diffusion de test ou à des administrateurs. Le choix de l’option Envoi du mailing n’est plus disponible. De cette façon, il n’est pas possible d’envoyer plusieurs fois le même mailing par erreur. Le rapport d’état de distribution qui se trouve en bas de l’écran affiche le nombre de destinataires à qui le mailing a déjà été envoyé. Chaque fois que vous cliquez sur l’onglet Envoi du mailing, vous générez un rapport d’état. Le système n’effectue pas de mises à jour automatiques.
