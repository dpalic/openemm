Mailings - Statistiques

L’aperçu statistique vous donne la liste de toutes les lettres d’information envoyées jusqu’à présent, dans la section Statistiques mailings. Cliquez simplement sur une des lettres d’information se trouvant dans la colonne Mailings afin de la soumettre à l’évaluation. Vous verrez immédiatement le nombre de clics effectués sur les liens de votre mailing sous forme graphique et numérique. Vous pouvez accéder à cette même évaluation en sélectionnant un mailing dans la section Mailings et en cliquant sur l’onglet Statistiques.
Le bouton Exportation enregistre les données affichées sur votre disque dur dans un fichier CSV.

Vous obtenez immédiatement un aperçu des taux de clics nets et bruts pour tous les suivis de liens intégrés dans votre mailing. Ce taux inclut toutes les répétitions de clics effectuées lorsque le destinataire clique plus d’une fois sur un même lien. Le taux de clic net est indiqué entre parenthèses. Il indique le nombre de destinataires ayant cliqué sur le lien. Les clics répétés par le même destinataire n’entrent pas en ligne de compte.

  • Groupe cible: en sélectionnant le groupe cible, vous décidez ainsi des destinataires pour lesquels vous désirez obtenir des statistiques. Vous avez même la possibilité de comparer jusqu’à 5 groupes cibles en fonction de leurs statistiques en ajoutant plusieurs groupes cibles via le bouton Ajouter. Pour supprimer un groupe cible, cliquez sur le bouton Supprimer qui se trouve à côté du groupe cible que vous désirez supprimer.

    Vous pouvez aussi obtenir des informations supplémentaires sur tous les Courriels qui ont été ouverts. Un simple clic sur le lien Courriels ouverts et OpenEMM affichera les données en fonctions des domaines des adresses électroniques des destinataires. Le système classera ensuite les 20 premiers domaines en fonction du nombre de destinataires.
  • Courriels ouverts: ce numéro n’est valable que pour les Courriels HTML dont l’option « Calcul du taux d’ouverture » est active.
  • Exclusion : vous donne le nombre de destinataires qui ont utilisé le lien d’exclusion du mailing.
  • Retours : vous indique le nombre de Courriels dont l’adresse du destinataire était erronée ou sujettes à d’autres problèmes lors de la réception ce Mailing.
  • Destinataire: nombre de destinataires.
