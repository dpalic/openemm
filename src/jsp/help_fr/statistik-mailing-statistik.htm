Statistiques – Statistiques sur les mailings

Vous obtenez immédiatement un aperçu des taux de clics nets et bruts pour tous les liens compris dans votre mailing. Ce taux inclut toutes les répétitions de clics effectuées lorsque le destinataire clique plus d’une fois sur un même lien. Le taux de clic net est indiqué entre parenthèses. Il répertorie le nombre de destinataires ayant cliqué sur le lien. Les clics répétés par le même destinataire n’entrent pas en ligne de compte.
