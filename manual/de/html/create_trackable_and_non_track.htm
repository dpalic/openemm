Messbare und nicht messbare Links

Im OpenEMM dürfen Sie vor dem Versand festlegen, welche Links Sie messen (also in rdir-Links umwandeln) möchten und welche nicht. Für den Link im Beispiel schalten Sie die Messfunktion ab.

1.Dazu klicken Sie auf das Register Messbare Links. Es erscheint unter der Spalte URL eine Übersicht aller Links, die in den Textbausteinen vorhanden sind.
2.Um einen bestimmten Link zu bearbeiten klicken Sie den Text bei URL an oder das Symbol Bearbeiten am Ende der Zeile.
3.Der Eingabedialog für messbare Links zeigt ganz oben den ausgewählten Link an. Darunter finden Sie ein Eingabefeld für eine Beschreibung. Geben Sie hier einen knappen Text ein, der die Bedeutung des Links kurz beschreibt. Dieser Text bleibt für die Empfänger der E-Mail unsichtbar. Bei der statistischen Auswertung der Klicks zeigt der OpenEMM die Beschreibung aber an. Das erleichtert die Auswertung bei langen URLs erheblich. Wenn für einen messbaren Link keine Beschreibung angegeben wurde, wird zum besseren Überblick der im HTML-Inhalt angegebene Titel des Links (falls vorhanden)  in der Übersicht unter Beschreibung angezeigt.

Abb. 3.31: In diesem Eingabedialog stellen Sie ein, in welchen Mail-Formaten der Link messbar ist.

4.Die Auswahlliste Messbar legt fest, in welchem Mail-Format der Link in einen messbaren rdir-Link umgewandelt wird.
Bitte beachten Sie: Durch diese Einstellung nur HTML-Version ist der Link in der Textversion nicht mehr messbar und damit wertlos. Sie sollten diese Option also nur verwenden, wenn ein bestimmter Link unbedingt im Klartext erscheinen soll, etwa die allgemeine Firmenadresse (www.firma.de).
Sollten Sie nicht messbar ausgewählt haben, ist es aus technischen Gründen nicht möglich dem Link eine Aktion zuzuweisen, daher wird diese Auswahl deaktiviert.
5.Wenn Sie den Status Messbar für alle Links auf einmal ändern möchten, können Sie das unter der Rubrik Alle Links ändern im Bereich Messbar tun. Wählen Sie den entsprechenden Status in der Auswahlliste aus und speichern die Änderung.
6.Die Auswahllisten Aktion spielen für das Beispiel keine Rolle. Details dazu finden Sie in Kapitel "Kundenprofile pflegen mit Aktionen".
7.Abschließend übergeben Sie die geänderte Link-Einstellung mit der Schaltfläche Speichern an das System. Jetzt erhalten die Empfänger des Mailings den aussagekräftigen Link http://www.schuhshop.org/sandalen.

Zusätzlich zu den Detaileinstellungen für jeden einzelnen messbaren Link können Sie im unteren Fensterbereich des Registers Messbare Links Einstellungen festlegen, die für sämtliche messbaren Links des Mailings gelten sollen.

•Messbar: Diese Option erfüllt die gleiche Funktion wie im obigen Beispiel beschrieben. Über die Auswahlliste Messbar legen Sie fest, in welchem Mail-Format der Link in einen messbaren rdir-Link umgewandelt wird. Mit der Einstellung nicht messbar ist keiner der Links des Mailing messbar. Durch die Einstellung nur Text-Version sind die messbaren Links in der HTML-Version nicht messbar, umgekehrt sind die Links mit der Einstellung nur HTML-Version in der Textversion nicht messbar. Sollen die Links Ihres Mailings unabhängig von der Version messbar sein, wählen Sie die vierte Option Text- und HTML-Version.

Abb. 3.32: Die Option Messbar legt fest, in welchem Mailformat die Links messbar sind.

•Standard-Aktion: Mit dieser Option legen Sie fest, ob der OpenEMM für jeden messbaren Link Ihres Mailing eine oder mehrere standardisierte Aktionen ausführen soll. Damit lässt sich beispielsweise das Empfängerprofil so modifizieren, dass Sie sofort ablesen können, wer welches Mailing geöffnet beziehungsweise auf die darin enthaltenen messbaren Links geklickt hat. Bitte beachten Sie, dass die Parameter der Standard-Aktion nur mit messbaren Links funktionieren.
Der OpenEMM bietet Ihnen drei Einstellmöglichkeiten für Standard-Aktionen an:
Die Standard-Aktion verknüpft alle messbaren Links mit einer bestimmten Aktion. Die Aktion lässt sich pro Link individuell ändern. Mehr zu den Aktionen lesen Sie in Kapitel "Kundenprofile pflegen mit Aktionen".
Standard-Aktion bei Öffnung: Öffnet ein Empfänger das Mailing, wird die hier festgelegte Aktion ausgeführt.
Standard-Aktion bei Klick: Klickt ein Empfänger auf einen messbaren Link im Mailing, wird die hier festgelegte Aktion ausgeführt.

Abb. 3.33: Mit der Standard-Aktion legen Sie bestimmte Aktionen für alle messbaren Links des Mailings fest.
