Deactivating and uninstalling

If you no longer need the plugin, or want to replace it with another version, you can deactivate it and subsequently uninstall it. To do so, game called the Plugin Manager and click on the plugin line or on the icon at the end of the line. In the following window select deactivate.

To uninstall a plugin that is no longer needed, click again on the corresponding line in the Plugin Manager. In the following window now select uninstall.
