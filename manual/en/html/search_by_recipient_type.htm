Search by recipient type

The search criterion Recipient type provides a convenient way of displaying all subscribers of a certain type. Depending on which type the recipient is set up as (see chapter "Mailing lists and recipient types"), the subscribers are differentiated as Administrator, Test recipient, Normal recipient. This function is used for your personal control: You can check here who is authorized to receive which type of newsletter and if necessary make corrections to the recipient data.
